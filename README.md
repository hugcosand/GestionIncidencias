# Descripción
El Sistema de Gestión de Incidencias IES SERPIS es una aplicación diseñada para digitalizar el registro y
seguimiento de incidencias relacionadas con el alumnado en un entorno educativo. Permite a profesores y personal
administrativo registrar, actualizar, visualizar y filtrar incidencias de manera eficiente, incorporando además
funcionalidades de gestión de usuarios, calendario del profesorado y notificaciones.
